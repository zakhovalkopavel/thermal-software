# thermal-software
Some thermal calculation utilities. For now:
- calculating quenching process. Main process it is intensive quenching during nucleate boiling. But it is also calculations 
  for oils and timeline for core temperature changing.
  In the perspective I want to add visualisation, calculations by air cooling, and parsing of cooling curve diagrams for 
  building models of quenchants cooling fluxes. After I want to add parsing of CCT diagrams for automatic phases composition 
  calculations and prediction of steel conditions after quenching.
- calculating recuperator and burning process for coal furnace. There is several recuperator configurations. Goal is 
  to estimate influences of sizes, burning power, materials and geometrical constructions for selecting best and simplest
  model. It provides better fuel efficiency and higher burning temperature.

It is nice to add complete links for used resources and articles, and make refactoring. For now it is rough-and-ready for
fast experiments developing.
