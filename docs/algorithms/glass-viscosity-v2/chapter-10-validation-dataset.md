## Overview

This chapter contains the complete reference datasets against which the implementation must be validated. Two independent datasets are provided:

1. **Lakatos dataset** ג€” 44 glass compositions with model-predicted isokom temperatures and residuals (־”T = calculated גˆ’ measured)

## Lakatos 1976 ג€” Validation Dataset

Source: `page_003_table_001.csv` (30 glasses) + `page_004_table_002.csv` (14 glasses)

### Dataset Structure

Each row gives:
- Composition in **wt%** (raw input)
- Calculated temperature at log ־· 2, 4, 6 poise
- ־”T = Calculated גˆ’ Measured (positive = model higher than experiment)

### Acceptance Criterion

For each row, the implementation must reproduce the **Calculated T** values (not the measured values):

```
|T_implementation(v) גˆ’ T_paper(v)| < 1.0ֲ°C for each viscosity level v
```

This tests the coefficient evaluation and encoding, not the model accuracy.

The ־”T values in the paper document the original model's accuracy against the training measurements. The reported standard deviations are ֿƒג‚‚ = 4.63ֲ°C, ֿƒג‚„ = 3.34ֲ°C, ֿƒג‚† = 3.14ֲ°C.
