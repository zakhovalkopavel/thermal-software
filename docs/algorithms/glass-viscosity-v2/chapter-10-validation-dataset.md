# Chapter 10: Validation Dataset

**Part IV: Validation**

---

## Overview

This chapter contains the complete reference datasets against which the implementation must be validated. Two independent datasets are provided:

1. **Lakatos dataset** — 44 glass compositions with model-predicted isokom temperatures and residuals (ΔT = calculated − measured)
2. **Fluegel dataset** — 6 NIST/DGG standard glasses with experimental vs model isokom temperatures

---

## Lakatos 1976 — Validation Dataset

Source: `page_003_table_001.csv` (30 glasses) + `page_004_table_002.csv` (14 glasses)

### Dataset Structure

Each row gives:
- Composition in **wt%** (raw input)
- Calculated temperature at log η 2, 4, 6 poise
- ΔT = Calculated − Measured (positive = model higher than experiment)

### Acceptance Criterion

For each row, the implementation must reproduce the **Calculated T** values (not the measured values):

```
|T_implementation(v) − T_paper(v)| < 1.0°C for each viscosity level v
```

This tests the coefficient evaluation and encoding, not the model accuracy.

The ΔT values in the paper document the original model's accuracy against the training measurements. The reported standard deviations are σ₂ = 4.63°C, σ₄ = 3.34°C, σ₆ = 3.14°C.

## VTF Round-Trip Validation

After fitting VTF to the three isokom points, the fitted curve must reproduce the input points:

### Criterion

For each Lakatos glass, after encoding → isokom → VTF fit, evaluate the VTF at the original three viscosity levels:

```
|T_VTF(logEta_i) − T_isokom_i| < 0.01°C  for i = 1, 2, 3
```

This is guaranteed analytically (the 3-point solve is exact) but should be verified numerically.

For Fluegel glasses, the same criterion applies for the three Fluegel viscosity levels.
