About Refractory Calculator Suite

Overview

The Refractory Calculator Suite is a comprehensive computational toolset for designing and analyzing refractory materials. Built with TypeScript and modern web technologies, it provides advanced calculations for phase equilibrium, particle size distribution optimization, and thermal performance prediction.

Modules

Phase Equilibrium Calculator

Calculate liquid and solid phase distributions at high temperatures using thermodynamically accurate models. Includes mineral phase identification, glass viscosity analysis, and refractoriness predictions according to international standards (EN ISO 1893, ASTM C24/C71, GOST 4069-69).

Polyfractional Blend Optimizer

Design optimal particle size distributions using Andreasen and Funk-Dinger models. Calculate packing properties with CPM (de Larrard) and Furnas methods. Predict chemical and sintering shrinkage. Save and manage custom mix presets in your personal library.

Technical Approach

All calculations are based on peer-reviewed literature and established standards:

  • Thermodynamic phase equilibrium using ACerS phase diagrams
  • Glass viscosity models (Giordano et al. 2008, Mills & Sridhar 1999)
  • Particle packing (de Larrard 1999, Furnas 1931, Funk & Dinger 1994)
  • Sintering kinetics (Master Sintering Curve, Coble diffusion model)
  • Refractoriness standards (EN ISO 1893, ASTM, GOST)
